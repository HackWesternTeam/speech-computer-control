Operating Computer with Google Speech API

Simon Yan & Pratham Thukral

Description

We had noticed a significant lack of accessibility features in some of the most common devices used: computers. We did some research on other methods that could help people with accessiblilty issues. We came across solutions such as embedding keyboards in peoples skin, using high speed (expensive) infrared cameras to track eye movement, or facial movement detection. However all of these felt too invasive or too complicated.

With the help of Google's Cloud Speech recognition API and open source code, we were able to create a voice-recognition software to help detect voice commands for a computer.

Cheat Sheet

Voice Command Description
open webpage Opens a webpage. Currently only supports: http://www.{webpage}.com
search Google for search query Searches Google in a new tab for your query
search Youtube for search query Searches Youtube in a new tab for your query
switch Switches windows ≈ alt+tab
back Goes back a page in history in Chrome
forward Goes forward a page in history in Chrome
close tab Closes current chrome tab ≈ ctrl+w
reopen tab Repoens recently closed tab ≈ ctrl+shift+t
close tab number Closes a specific chrome tab
go to tab number Goes to a specific tab number
type text Starts typing wherever your cursor is
scroll up scrolls up
scroll down scrolls down
send Used in messenger applications ≈ enter
copy Selects all and copies ≈ ctrl+a then ctrl+c
paste Pastes selection ≈ ctrl+v
clear Clear all text ≈ ctrl+a then backspace
set volume to number Sets system volume to a number
l number Moves the mouse cursor left in 25 pixel increments
r number Moves the mouse cursor right in 25 pixel increments
u number Moves the mouse cursor up in 25 pixel increments
d number Moves the mouse cursor down in 25 pixel increments
click Initiates a click event where the cursor is
start Simulates Start/Windows key press
open application application name Opens the application of choice
zoom in percentage Zooms into the window to a certain percentage
zoom out percentage Zooms out of the window to a certain percentage
close window Closes current window ≈ alt+f4
minimize window Minimizes current window
fullscreen Toggles full screen mode on YouTube
exit Stops the speech-to-text process
